# Digital Sanctuary Quotes

Public JSON data source for the [Digital Sanctuary](https://github.com/sudarshanawasthi47/digital-sanctuary) motivational quote app.

## Structure

```json
[
  {
    "id": 1,
    "quote": "Discipline is choosing between what you want now and what you want most.",
    "author": "Abraham Lincoln",
    "category": "Discipline"
  }
]
```

## Categories

- Discipline
- Wisdom
- Courage
- Love
- Success
- Happiness
- Motivation
- Leadership
- Creativity
- Peace
- Growth
- Resilience
- Mindfulness
- Gratitude
- Perseverance
- Focus
- Kindness
- Strength
- Hope
- Change

## Usage

Raw URL for fetching:

```
https://raw.githubusercontent.com/sudarshanawasthi47/digital-sanctuary-quotes/main/quotes.json
```

## Contributing

To add new quotes, submit a PR with new entries appended to `quotes.json`. Each quote must have a unique `id`, and the `category` should match one of the existing categories above (or propose a new one).
